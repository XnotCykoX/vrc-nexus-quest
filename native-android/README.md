# Native Android sources (source of truth)

The `android/` folder is generated by `npx cap add android` and is git-ignored, so the
**custom native code lives here** and must be copied in if `android/` is ever recreated.

Copy these into `android/app/src/main/java/com/deize/vrcnexus/quest/`:
- `OscPlugin.java` — UDP socket bridge for OSC (send to VRChat :9000, listen on :9001)
- `MainActivity.java` — registers the OSC plugin

And ensure `android/app/src/main/AndroidManifest.xml` keeps:
```xml
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" android:maxSdkVersion="32" />
```
(INTERNET is already present and is what OSC/UDP + the VRChat API need.)

After copying: `npm run cap:sync` then rebuild.
